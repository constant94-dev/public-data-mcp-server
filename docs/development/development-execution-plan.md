# Public Data MCP Server - 실행 계획

## 🎯 프로젝트 목표

공공데이터포털 API를 활용하는 MCP 서버를 개발하여 AI 도구에서 공공데이터를 쉽게 활용할 수 있도록 합니다.

## 🏗️ 기술 스택

### 현재 사용 중
- **Java**: 21
- **Gradle**: 8.8
- **MCP Java SDK**: 0.11.1
- **Docker**: 컨테이너 지원

### 추가 예정
- **Spring WebClient**: HTTP 클라이언트
- **Redis**: 캐싱 시스템
- **Jackson**: JSON 처리
- **SLF4J + Logback**: 로깅
